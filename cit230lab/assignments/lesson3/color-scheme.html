Color Scheme

#212026

#F2E313

#A3B4BF

Background

The background color will be #212026 which is a dark color, this will make it easy to display weather information.

Headers

The color for headers will be #F2E313 which is a yellow color that compliments the background color nicely.

Paragraphs

The color for the paragraph texts is #A3B4BF. This is nice soft color and is easy to read with the background.
